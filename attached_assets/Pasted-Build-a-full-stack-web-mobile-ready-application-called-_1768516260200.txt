3) Backend (Node + Express API)

Build REST API with proper folder structure:

API Requirements

✅ Authentication Routes:

POST /api/auth/register

POST /api/auth/login

✅ Vehicle Routes:

POST /api/vehicles/add

GET /api/vehicles/my

GET /api/vehicles/all (admin)

PATCH /api/vehicles/:id/approve (admin)

PATCH /api/vehicles/:id/reject (admin)

PATCH /api/vehicles/:id/block (admin)

✅ Temporary Pass Routes:

POST /api/passes/create

GET /api/passes/my

GET /api/passes/all (admin)

✅ Gate Logs Routes:

POST /api/logs/create (used by ANPR gate system)

GET /api/logs/my

GET /api/logs/all (admin)

✅ ANPR Gate Verification Route:

POST /api/gate/verify
Input: { plateNumber: "MP09AB1234" }
Output:

if approved / active pass found → { allowed: true, reason: "approved_vehicle" }

else → { allowed: false, reason: "not_found" }
